Reference Implementations

Reference implementations and test suites for the routines in Fortran95, Fortran77, and C, are in preparation.

  • Reference implementation for the Fortran 95 interface to the Sparse BLAS (December, 2000)
  • Reference implementation for the C interface to the Extended and Mixed Precision BLAS (January, 2001)
  • Reference implementation for the C interface to the Legacy BLAS (March, 1999)
    • cblas.h
  • Reference implementation for the Fortran95 interface to the Legacy BLAS

Modules and Include Files

Fortran 95

  • blas_dense.f90 -- Fortran 95 module for Chapter 2
    • blas_operator_arguments.f90 -- Fortran 95 module of derived types for Chapters 2 and 4
    • blas_precision.f90 -- Fortran 95 module of precisions for Chapters 2 and 4
    • blas_dense_red_op.f90 -- Fortran 95 module of reduction operations routines for Chapter 2
    • blas_dense_gen_trans.f90 -- Fortran 95 module of generate transformations routines for Chapter 2
    • blas_dense_vec_op.f90 -- Fortran 95 module of vector operations routines for Chapter 2
    • blas_dense_vec_mov.f90 -- Fortran 95 module of data movement with vectors routines for Chapter 2
    • blas_dense_mat_vec_op.f90 -- Fortran 95 module of matrix-vector operations routines for Chapter 2
    • blas_dense_mat_op.f90 -- Fortran 95 module of matrix operations routines for Chapter 2
    • blas_dense_mat_mat_op.f90 -- Fortran 95 module of matrix-matrix operations routines for Chapter 2
    • blas_dense_mat_mov.f90 -- Fortran 95 module of data movement with matrices routines for Chapter 2
    • blas_dense_fpinfo.f90 -- Fortran 95 module of environmental enquiry routine for Chapter 2
  • blas_sparse.f90 -- Fortran 95 module for Chapter 3
    • blas_sparse_namedconstants.f90 -- Fortran 95 module of named constants for Chapter 3
  • blas_extended.f90 -- Fortran 95 module for Chapter 4
    • blas_extended_red_op.f90 -- Fortran 95 module of reduction operations routines for Chapter 4
    • blas_extended_vec_op.f90 -- Fortran 95 module of vector operations routines for Chapter 4
Fortran 77
  • blas_namedconstants.h -- Fortran 77 include file for the values of named constants
C
  • blas.h -- C include file containing the value of enumerated types and all prototypes for Chapters 2, 3, and 4
  • blas_dense.h -- C include file for Chapter 2 (containing blas_enum.h + blas_dense_proto.h)
  • blas_sparse.h -- C include file for Chapter 3 (containing blas_enum.h + blas_malloc.h + blas_sparse_proto.h)
  • blas_extended.h -- C include file for Chapter 4 (containing blas_enum.h + blas_malloc.h + blas_extended_proto.h)
  • blas_enum.h -- C include file for the value of enumerated types
  • blas_malloc.h -- C include file for the BLAS malloc functions
  • blas_dense_proto.h -- C include file containing the prototypes for Chapter 2
  • blas_sparse_proto.h -- C include file containing the prototypes for Chapter 3
  • blas_extended_proto.h -- C include file containing the prototypes for Chapter 4
